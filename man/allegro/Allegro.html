Module Allegro


module Allegro: sig .. end
This is an OCaml binding for Allegro. Allegro is a cross-platform library intended for use in computer games and other types of multimedia programming.

  • General Functions
  • Graphics modes
  • Truecolor pixel formats
  • Palette routines
  • Converting between color formats
  • Bitmap objects
  • Loading image files
  • Keyboard routines
  • Timer routines
  • Mouse routines
  • Fixed point number
  • Drawing primitives
  • Sprites
  • RLE Sprites
  • Compiled Sprites
  • Text output
  • Transparency and patterned drawing
  • Sound init routines
  • Digital sample routines
  • File and compression routines
  • Datafile routines
  • Polygon rendering
  • 3D math routines
  • Quaternion math routines
  • GUI routines
